# actproof-events 2.0.0 — Reviewed profile governance

This release introduces reviewed profile governance and a bank POC packaging surface.

## Added

- `actproof_events.profile_governance`
- `governance-status` CLI command
- `review-records` CLI command
- `challenge-records` CLI command
- `validate-governance` CLI command
- `export-bank-poc-pack` CLI command
- maintainer-reviewed DORA governance record
- public challenge/gap record registry
- bank POC pack export

## Boundaries

Maintainer-reviewed does not mean legal review, bank SME approval, supervisory approval or compliance certification. Candidate mapping remains candidate/review-required only.
