These artefacts demonstrate implementation experience with the SCITT-adjacent
substrate primitives that any IETF SCITT-conformant Transparency Service must
provide or interoperate with: deterministic canonicalization, hash-tree
proofs, external temporal anchoring, and profile-conformant evidence
records. The Quoruna application (https://quoruna.com) is the first
production consumer of both artefacts and provides end-to-end verification
flow visible in any issued receipt.
