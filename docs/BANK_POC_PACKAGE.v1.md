# ActProof bank POC package v1

The bank POC pack is a file-based bundle for local evaluation of one ActProof profile against one draft report and one internal field list.

It includes:

1. pinned profile lockfile
2. profile-view JSON
3. source-atom coverage report
4. completeness report
5. sample draft incident report
6. sample internal field list
7. candidate mapping output
8. pre-validation report
9. known boundaries
10. bank review checklist
11. governance status, review records and challenge records

The pack is intended for controlled internal evaluation. It is not a production incident-reporting workflow and not a compliance certification.
